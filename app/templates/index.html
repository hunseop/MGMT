🚀 MGMT 사용설명서 🚀

1. 시작하기

명령을 실행하기 위해서는 먼저 방화벽을 등록해야 합니다. 'Devices' 메뉴로 가서 방화벽을 등록하세요.

2. 장치 관리

장치를 추가하거나, 엑셀파일 업로드를 통해 일괄 등록하세요. 'Devices' 메뉴에서 모든 것이 가능합니다.

3. 업로드 기능

엑셀 파일을 통해 장치 정보를 손쉽게 일괄 등록할 수 있습니다. 'Upload Devices'를 클릭하고 파일을 선택하세요.

4. 장치 검색

필요한 장치를 빠르게 찾아보세요. 검색 기능을 통해 이름이나 IP로 장치를 찾을 수 있습니다.

5. 명령 보내기

선택한 장치에 명령을 보내십시오. 간단한 입력과 클릭으로 모든 것이 처리됩니다.
